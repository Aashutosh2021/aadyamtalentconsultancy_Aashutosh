# Project Documentation

## Objective

To develop a Machine Learning model capable of predicting house rent prices based on various housing features.

## Problem Statement

House rent prices vary depending on multiple factors such as size, location, furnishing status, and number of rooms. The objective is to estimate rent using these features.

## Data Preprocessing

The following preprocessing steps were performed:

* Dataset loading
* Feature selection
* Categorical data encoding
* Dataset cleaning

## Model Development

Linear Regression was selected because rent prediction is a regression problem involving continuous numerical output.

## Evaluation Metrics

* R² Score
* Mean Absolute Error (MAE)

## Results

R² Score: 0.4401

MAE: 24408.85

The model successfully predicts rental values and demonstrates the complete machine learning workflow.

## Future Improvements

* Feature Engineering

## Conclusion

The project successfully demonstrates data preprocessing, regression modeling, prediction, and evaluation using Machine Learning techniques.
