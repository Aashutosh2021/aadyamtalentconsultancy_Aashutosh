# House Rent Prediction using Machine Learning

## Project Overview

This project predicts house rent prices using Machine Learning Regression techniques.

The model is trained on a Kaggle House Rent dataset and uses property-related features such as:

* BHK
* Size
* Bathroom
* City
* Furnishing Status
* Tenant Preferred
* Area Type

## Dataset

Dataset: House Rent Dataset

Total Records: 4746

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib

## Workflow

1. Data Collection
2. Data Preprocessing
3. Feature Encoding
4. Model Training
5. Prediction
6. Model Evaluation
7. Visualization
