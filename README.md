# 🏠 Project 1 – House Rent Prediction

### Objective

Predict house rental prices based on property-related features.

### Features Used

* BHK
* Size
* Bathroom
* City
* Area Type
* Furnishing Status
* Tenant Preference

### Algorithm

* Linear Regression

### Performance

* R² Score : **0.4401**
* MAE : **24408.85**

### Deliverables

* Data Preprocessing
* Trained Model (.pkl)
* Prediction Results
* Visualizations
* Documentation

# 🚗 Project 2 – Car Price Prediction

### Objective

Estimate used car selling prices using machine learning.

### Features Used

* Year
* KM Driven
* Fuel Type
* Seller Type
* Transmission
* Owner
* Mileage
* Engine
* Max Power
* Seats

### Algorithm

* Random Forest Regressor

### Performance

* RMSE : **132840.30**

### Deliverables

* Prediction Results
* Feature Importance Graph
* Actual vs Predicted Visualization
* Trained Model

# 📰 Project 3 – Fake News Detection System

### Objective

Develop an NLP model capable of identifying fake and real news articles.

### Technologies

* TF-IDF Vectorization
* Logistic Regression

### Performance

* Accuracy : **98.55%**
* Precision : **98.29%**
* Recall : **98.66%**
* F1 Score : **98.47%**

### Deliverables

* Fake News Classifier
* TF-IDF Vectorizer
* Prediction Results
* Confusion Matrix
* Classification Report
